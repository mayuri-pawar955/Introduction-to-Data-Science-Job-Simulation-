# Commonwealth Bank – Introduction to Data Science (Forage)
# Overview
This repository contains my completed work for the Commonwealth Bank – Introduction to Data Science virtual experience program on Forage.
The project focuses on data management, data exploration, data transformation, and analytical thinking, simulating real-world tasks performed by Commonwealth Bank’s Data Science team.


# Program Details
- Organization:Commonwealth Bank
- Platform:Forage
- Program: Introduction to Data Science
- Domain: Data Management & Analytics
